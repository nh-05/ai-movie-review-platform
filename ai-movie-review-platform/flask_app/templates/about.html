About CineAI

Flask-powered movie review backend

🐍 Flask Backend

This Flask application serves as the REST API backend for the CineAI platform. It provides movie data, routes for the frontend, and integrates with the FastAPI sentiment service and Django blog system.

  • Python 3.10+
  • Flask 3.x
  • Flask-CORS for cross-origin support
  • JSON-based movie dataset
