What is CineAI?

CineAI is a comprehensive movie review platform that combines traditional web technologies with modern AI capabilities. Users can browse movies, read and write reviews, and get instant AI-powered sentiment analysis on any review text.

The platform demonstrates a complete full-stack architecture spanning HTML/Bootstrap frontend, Flask REST API, FastAPI ML service, and a Django-powered blog system.
